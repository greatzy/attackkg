## 1. 系统概述

### 1.1 系统简介

网络安全攻击知识库系统是一个基于 MITRE ATT&CK 框架的威胁知识管理平台，帮助安全团队：

- **了解攻击技术**：系统化学习 MITRE ATT&CK 框架的战术和技术
- **可视化分析**：通过矩阵和图谱直观展示攻击路径
- **威胁情报管理**：整合威胁行为者信息和检测规则
- **知识共享**：支持团队协作和知识沉淀

### 1.2 系统特点

- **数据全面**：包含 14 个战术、691 个技术、187 个威胁行为者
- **可视化强**：ATT&CK 矩阵、攻击路径图谱、统计报表
- **易于使用**：直观的 Web 界面，无需编程基础
- **权限控制**：基于角色的访问控制，保障数据安全

### 1.3 系统访问

**访问地址**：http://localhost:5174

**支持浏览器**：
- Google Chrome 90+
- Mozilla Firefox 88+
- Microsoft Edge 90+

**推荐分辨率**：1920×1080 或更高

## 2. 快速开始

### 2.1 登录系统

1. 打开浏览器，访问 http://localhost:5174
2. 在登录页面输入用户名和密码
   - 默认管理员账号：admin / admin123
   - 默认测试账号：testuser / testpassword
3. 点击【登录】按钮进入系统

![登录页面](images/login.png)

### 2.2 系统主界面

登录成功后，您将看到系统主界面：

![主界面](images/main-interface.png)

**界面组成**：
- **顶部导航栏**：显示系统名称、用户信息、退出按钮
- **左侧菜单栏**：功能模块导航
- **主内容区**：显示当前功能页面

### 2.3 修改密码

1. 点击右上角用户名
2. 选择【修改密码】
3. 输入原密码和新密码
4. 点击【确认】保存

## 3. 功能模块使用指南

### 3.1 仪表板

**功能说明**：展示系统概览和关键统计数据

**访问路径**：首页 > 仪表板

**主要功能**：
- 查看 ATT&CK 数据统计（战术数、技术数、子技术数）
- 查看最新威胁情报
- 查看系统活动日志

**操作步骤**：
1. 点击左侧菜单【首页】
2. 查看各项统计数据
3. 点击卡片可跳转到对应模块

### 3.2 ATT&CK 知识库

#### 3.2.1 战术管理

**功能说明**：管理 MITRE ATT&CK 的 14 个战术类别

**访问路径**：ATT&CK 知识库 > 战术管理

**主要功能**：
- 查看战术列表
- 搜索战术
- 查看战术详情

**操作步骤**：
1. 点击左侧菜单【ATT&CK 知识库】
2. 展开子菜单，点击【战术管理】
3. 在列表中查看所有战术
4. 使用搜索框搜索特定战术
5. 点击战术名称查看详情

#### 3.2.2 技术管理

**功能说明**：管理 MITRE ATT&CK 的 691 个技术

**访问路径**：ATT&CK 知识库 > 技术管理

**主要功能**：
- 查看技术列表
- 按战术筛选
- 按平台筛选
- 查看技术详情

**操作步骤**：
1. 点击【技术管理】
2. 使用筛选条件：
   - 选择【战术】下拉框按战术筛选
   - 选择【平台】下拉框按平台筛选
   - 勾选【仅显示子技术】筛选子技术
3. 在列表中查看技术
4. 点击技术名称查看详情，包括：
   - 技术描述
   - 适用平台
   - 相关缓解措施
   - 相关威胁行为者

#### 3.2.3 缓解措施管理

**功能说明**：查看和管理防御措施

**访问路径**：ATT&CK 知识库 > 缓解措施

**主要功能**：
- 查看缓解措施列表
- 查看缓解措施详情
- 查看关联的技术

### 3.3 威胁行为者

**功能说明**：管理已知的威胁行为者（APT 组织）信息

**访问路径**：威胁行为者

**主要功能**：
- 查看威胁行为者列表
- 搜索行为者
- 查看行为者详情
- 查看行为者使用的技术

**操作步骤**：
1. 点击左侧菜单【威胁行为者】
2. 在列表中查看所有威胁行为者
3. 使用搜索框搜索特定行为者
4. 点击行为者名称查看详情，包括：
   - 行为者简介
   - 别名
   - 来源地区
   - 使用的技术列表
   - 活动时间线

### 3.4 检测规则

**功能说明**：管理安全检测规则

**访问路径**：检测规则

**主要功能**：
- 查看规则列表
- 创建新规则
- 编辑规则
- 删除规则
- 验证规则
- 导出规则

**创建规则步骤**：
1. 点击【检测规则】
2. 点击【新建规则】按钮
3. 填写规则信息：
   - 规则名称
   - 规则类型（Sigma、YARA、Snort 等）
   - 关联技术
   - 适用平台
   - 严重级别
   - 规则内容
4. 点击【保存】

**验证规则**：
1. 在规则列表中找到目标规则
2. 点击【验证】按钮
3. 系统会检查规则语法
4. 查看验证结果

### 3.5 可视化分析

#### 3.5.1 ATT&CK 矩阵

**功能说明**：以矩阵形式展示 ATT&CK 框架的战术和技术关系

**访问路径**：可视化分析 > ATT&CK 矩阵

**主要功能**：
- 查看完整的 ATT&CK 矩阵
- 按平台筛选技术
- 查看技术详情
- 刷新数据

**操作步骤**：
1. 点击左侧菜单【可视化分析】
2. 点击【ATT&CK 矩阵】
3. 查看矩阵：
   - 横向：14 个战术类别
   - 纵向：每个战术下的技术
4. 使用平台筛选器：
   - 选择【Windows】只显示 Windows 平台的技术
   - 选择【Linux】只显示 Linux 平台的技术
   - 以此类推
5. 点击技术卡片查看详情
6. 点击【刷新数据】获取最新数据

**界面说明**：
- 蓝色卡片：战术
- 绿色卡片：主技术
- 橙色卡片：子技术

#### 3.5.2 攻击路径图谱

**功能说明**：以图谱形式展示攻击路径和技术关系

**访问路径**：可视化分析 > 攻击路径图谱

**主要功能**：
- 查看攻击路径图谱
- 按战术筛选
- 查看节点详情
- 分析攻击路径
- 控制图谱动画

**操作步骤**：
1. 点击【攻击路径图谱】
2. 查看图谱：
   - 节点：战术（蓝色）、技术（绿色）、子技术（橙色）
   - 边：归属关系（实线）、攻击路径（虚线）
3. 交互操作：
   - **点击节点**：查看节点详情和攻击路径分析
   - **拖拽节点**：调整节点位置
   - **滚轮缩放**：放大/缩小图谱
   - **拖拽画布**：移动视图
4. 使用控制按钮：
   - 【重置视图】：恢复初始视图
   - 【开启动画/停止动画】：控制物理动画
5. 使用战术筛选器查看特定战术的图谱

**路径分析**：
1. 点击某个技术节点
2. 在右侧【攻击路径分析】面板中查看：
   - 从该技术开始的攻击路径
   - 时间线形式展示

### 3.6 威胁情报

**访问路径**：威胁情报

## 4. 常见问题

### 4.1 登录问题

**Q1: 忘记密码怎么办？**

A: 请联系系统管理员重置密码。

**Q2: 登录时提示"用户已被禁用"？**

A: 您的账号已被管理员禁用，请联系管理员了解详情。

**Q3: Token 过期怎么办？**

A: 系统会自动刷新 Token，如果仍然提示过期，请重新登录。

### 4.2 功能使用问题

**Q4: 矩阵页面加载缓慢？**

A: 矩阵包含大量数据，首次加载可能需要几秒钟。您可以：
- 使用平台筛选减少数据量
- 检查网络连接
- 清除浏览器缓存

**Q5: 如何快速找到特定技术？**

A: 您可以使用以下方法：
- 在【技术管理】页面使用搜索功能
- 在【ATT&CK 矩阵】中按战术筛选
- 使用技术 ID（如 T1566）搜索

**Q6: 如何查看某个技术的所有子技术？**

A: 
1. 进入【技术管理】
2. 在技术列表中找到目标技术
3. 查看详情页面中的【子技术】部分

### 4.3 数据问题

**Q7: 数据如何更新？**

A: 系统支持从 MITRE 官方仓库导入最新数据：
1. 管理员进入【系统管理】
2. 点击【数据管理】
3. 选择【导入数据】
4. 选择数据源并导入

**Q8: 可以导出数据吗？**

A: 是的，大部分功能支持数据导出：
- 技术列表：导出为 Excel 或 JSON
- 检测规则：导出为规则文件
- 审计日志：导出为 CSV

### 4.4 权限问题

**Q9: 无法访问某些功能？**

A: 您的账号可能没有相应权限。请联系管理员：
- 确认您的角色权限
- 申请需要的权限

**Q10: 如何成为管理员？**

A: 现有管理员可以在【用户管理】中：
1. 找到您的账号
2. 编辑用户信息
3. 分配管理员角色

## 5. 附录

### 5.1 术语表

| 术语 | 英文 | 说明 |
|------|------|------|
| 战术 | Tactic | 攻击者的战术目标，如 Initial Access（初始访问） |
| 技术 | Technique | 攻击者实现战术目标的具体方法，如 Phishing（钓鱼） |
| 子技术 | Sub-technique | 技术的细分变体，如 Spearphishing Attachment（鱼叉式钓鱼附件） |
| 缓解措施 | Mitigation | 防御或减轻攻击技术的方法 |
| 威胁行为者 | Threat Actor | 实施攻击的组织或个人，如 APT28 |
| IOC | Indicator of Compromise | 失陷指标，如恶意 IP、域名等 |
| TTP | Tactics, Techniques, and Procedures | 战术、技术和程序 |
