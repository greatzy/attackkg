# 网络安全攻击知识库系统 - 项目简介

## 项目概述

网络安全攻击知识库系统（AttackKg）是一个基于MITRE ATT&CK框架的知识管理和可视化平台，帮助安全团队高效管理和分析攻击技术数据。

## 核心功能

| 功能模块 | 功能描述 |
|----------|----------|
| 知识管理 | 战术、技术、子技术、缓解措施的管理和查询 |
| 可视化分析 | 攻击矩阵、攻击路径图谱、关联分析 |
| 数据导入 | 支持MITRE官方STIX格式数据导入 |
| 用户管理 | 基于RBAC的权限控制系统 |

## 技术架构

- **前端**：Vue 3 + TypeScript + Element Plus + ECharts
- **后端**：Flask + SQLAlchemy + JWT
- **数据库**：SQLite（开发）/ PostgreSQL（生产）

## 提效成果

在TRAE SOLO辅助下，原本需要10-13天的开发工作缩短至3.5天，提效达70%。

## 适用场景

- 安全分析与威胁情报管理
- 安全培训和教育
- 渗透测试报告生成
- 企业安全评估
